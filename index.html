Experiences and Projects:

Complex Network Analysis

  • Stanford's Amazon product co-purchasing network dataset
    • Nodes are the items on sale on the website
    • Edges are between those nodes which correspond to items that are frequently bought together
  • Analysed the network by:
    • Performing network clustering using modularity maximization
    • Evaluating metrics such as the clustering coefficient, assortativity, mean geodesic path lenght and fraction of vertices in the largest component
  • I can share a few insights that we obtained from the analysis:
  • The size of the largest component is 1 which means that any two products can be shown to be similar to each other though a chain of other nodes
  • The value of the average clustering coefficient is 0.38 which is quite high and this can be attributed to the fact that there are several products like groceries/electronics products which are frequently bought together with a range of different products
  • Mean shortest path length is 11.73, this hints towards the fact that there are products which create links between products of entirely different categories
